Memory Game

About

This Memory based Card Game is a fun and interactive game that exercises the brain and improves concentration. It helps in developing strong observation skills in children.

How To Play?

This memory game is very easy to play with. On each turn, a player turns over any two cards (one at a time) and keeps them if the cards match. To move ahead the player needs to match symbols with the cards. The game can be played solo or with friends. Try to match all the cards in the fastest time possible.

Dots And Boxes

About

Dots and Boxes is a fun and simple classic pen-and-paper game that exercises the thinking abilities and visual observational skills.

How To Play?

Each turn, click between two horizontally or vertically adjacent dots to draw a line Drawing the 4th wall of a box wins it, earning you a point. When you close a box you must move again. Lines are drawn until all squares are claimed. The player with the most claimed squares wins!
